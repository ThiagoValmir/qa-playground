# QA Playground - Login & Register Tests

Projeto de testes manuais e automatizados para os módulos de Login e Registro de Usuários.

## Objetivos
- Validar fluxo de autenticação
- Validar fluxo de cadastro
- Executar testes manuais
- Automatizar cenários críticos

## Ferramentas
- Cypress
- Javascript
- GitHub Actions
